    Software de Escritorio

    Sistema de inventario para el casino de Duoc UC utilizando Java y el patrón de diseño MVC (Modelo-Vista-Controlador). Este sistema se implementó en un entorno de desarrollo integrado Apache NetBeans y se conectó a una base de datos MySQL a través de MySQL Workbench. La interfaz gráfica del sistema está construida con Java Swing, con un JFrame principal que presenta un formulario único. Este formulario permite realizar diversas operaciones en la tabla de inventario. El sistema ofrece las siguientes funcionalidades: 1. Agregar Producto: Permite registrar nuevos productos con validaciones integradas para evitar inconsistencias en los datos. 2. Modificar Producto: Permite buscar y actualizar productos existentes en la base de datos. 3. Eliminar Producto: Permite eliminar un producto después de solicitar confirmación al usuario. 4. Listar Productos: Presenta todos los productos en un JTable, con la capacidad de filtrar por categoría para una fácil búsqueda. El diseño del sistema sigue el patrón de diseño MVC, separando claramente la lógica de negocio (Modelo), la presentación de datos (Vista) y el control de las interacciones del usuario (Controlador). Esta arquitectura facilita el mantenimiento y la escalabilidad del sistema. El sistema ha sido desarrollado con Java orientado a objetos, haciendo uso de las mejores prácticas de programación para garantizar la eficiencia y la robustez del código.

    Proyecto ConectArte

    En mi proyecto de UX diseñé una aplicación móvil innovadora que conecta artistas locales con compradores de arte. Durante agosto y septiembre de 2024, me desempeñé como diseñadora UX principal, enfocándome en crear una plataforma que resuelve dos problemas principales: la dificultad de los artistas para exhibir y vender sus obras, y la necesidad de los compradores de encontrar arte local auténtico. Realicé una investigación profunda con ambos grupos de usuarios, desarrollé wireframes y prototipos, e implementé mejoras basadas en pruebas de usabilidad. La aplicación resultante ofrece una interfaz intuitiva que permite a los artistas gestionar sus obras fácilmente y a los compradores adquirir arte local con confianza. Incorporé consideraciones de accesibilidad como alto contraste y textos legibles, asegurando que la plataforma sea inclusiva para todos los usuarios. Este proyecto demuestra mi capacidad para diseñar soluciones centradas en el usuario que atienden necesidades reales del mercado.

    Proyecto VecindApp

    Como Diseñadora UX principal, lideré el desarrollo de VecindApp, una plataforma web innovadora diseñada para juntas de vecinos que participó en el prestigioso Torneo "ALL IN CHILE" organizado por Duoc UC. Nuestro proyecto se destacó entre 1,390 postulaciones, siendo seleccionado entre los 110 equipos finalistas y alcanzando una posición entre los 9 mejores equipos de la región. Durante agosto a octubre de 2024, me enfoqué en crear una solución que abordara los desafíos críticos en la comunicación y organización comunitaria. Realizé una investigación exhaustiva con miembros de juntas vecinales, desarrollé wireframes y prototipos, e implementé mejoras basadas en pruebas de usabilidad con usuarios reales. El resultado fue una plataforma intuitiva que centraliza la comunicación vecinal, facilita la gestión de emergencias y eventos comunitarios, y promueve la transparencia en proyectos barriales. La aplicación incorpora características de accesibilidad para asegurar que todos los vecinos puedan participar activamente en su comunidad. Este proyecto demuestra mi capacidad para diseñar soluciones tecnológicas que abordan necesidades sociales reales y generan impacto comunitario.

    Proyecto CheckApp

    Como Diseñadora UX principal, desarrollé CheckApp, una aplicación móvil para gestión y registro de asistencia a eventos, como parte de un proyecto integrado para las asignaturas de Programación de Aplicaciones Móviles y Arquitectura. Durante agosto a diciembre de 2024, lideré la investigación con organizadores de eventos y participantes, identificando la necesidad de automatizar procesos tradicionalmente manuales y propensos a errores. El proyecto se destacó por su implementación completa: desde el diseño UX hasta el desarrollo funcional en Visual Studio utilizando Angular/Ionic. Realicé pruebas de usabilidad que revelaron y permitieron resolver desafíos clave como la navegación entre secciones, visibilidad en dispositivos móviles y optimización de tiempos de carga. La aplicación final no solo existe como prototipo en Figma, sino que está disponible como APK descargable a través de nuestra landing page, permitiendo su uso real en dispositivos Android. Este proyecto demuestra mi capacidad para llevar un diseño desde su concepción hasta su implementación técnica completa, creando una solución que optimiza la gestión de eventos y mejora la experiencia tanto para organizadores como para participantes.

    Herramientas utilizadas: Figma, Visual Stidio, Andoid Studio
