    API CLIMA

    La API de Pronóstico del Tiempo es una aplicación web que permite a los usuarios obtener información meteorológica actualizada para cualquier ciudad del mundo. Utilizando la API de OpenWeatherMap, esta aplicación proporciona datos precisos sobre la temperatura actual, la sensación térmica, la humedad y la descripción del clima para una ciudad específica. La interfaz de usuario está diseñada con colores celestes para crear una experiencia visualmente atractiva y fácil de usar. Esta aplicación es útil para cualquier persona que necesite conocer el clima de una ubicación específica en tiempo real.

    Herramientas utilizadas: HTML, CSS, JavaScript

    Software de Escritorio

    Sistema de inventario para el casino de Duoc UC utilizando Java y el patrón de diseño MVC (Modelo-Vista-Controlador). Este sistema se implementó en un entorno de desarrollo integrado Apache NetBeans y se conectó a una base de datos MySQL a través de MySQL Workbench. La interfaz gráfica del sistema está construida con Java Swing, con un JFrame principal que presenta un formulario único. Este formulario permite realizar diversas operaciones en la tabla de inventario. El sistema ofrece las siguientes funcionalidades: 1. Agregar Producto: Permite registrar nuevos productos con validaciones integradas para evitar inconsistencias en los datos. 2. Modificar Producto: Permite buscar y actualizar productos existentes en la base de datos. 3. Eliminar Producto: Permite eliminar un producto después de solicitar confirmación al usuario. 4. Listar Productos: Presenta todos los productos en un JTable, con la capacidad de filtrar por categoría para una fácil búsqueda. El diseño del sistema sigue el patrón de diseño MVC, separando claramente la lógica de negocio (Modelo), la presentación de datos (Vista) y el control de las interacciones del usuario (Controlador). Esta arquitectura facilita el mantenimiento y la escalabilidad del sistema. El sistema ha sido desarrollado con Java orientado a objetos, haciendo uso de las mejores prácticas de programación para garantizar la eficiencia y la robustez del código.
